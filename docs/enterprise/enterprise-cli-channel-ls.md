# channel ls

Lists enterprise channels

### Synopsis

Lists all channels that have been created

### Usage
```bash
replicated enterprise channel ls [flags]
```


| Flag                  | Type   | Description |
|-----------------------|--------|-------------|
| `-h, --help` | | help for ls |

### Examples

```bash
replicated enterprise channel ls
ID                             NAME
1aUcp52Hcvval50e1gyYaoW5oUO    Some Big Bank
```
