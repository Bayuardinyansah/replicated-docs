To generate a service account:

1. Log in to the vendor portal, and select [**Team > Service Accounts**](https://vendor.replicated.com/team/serviceaccounts).
1. Select **New Service Account**. If one or more service accounts already exist, you can add more by selecting **New Service Account**.

  <img alt="Service Accounts Page" src="/images/service-accounts.png" width="600px"/>

  [View a larger version of this image](/images/service-accounts.png)

1. In the **New Service Account** dialog, enter a nickname the token and select the RBAC policy from the dropdown list. Optionally, check the box to limit the token to a read-only version of the selected RBAC policy. Select **Create Service Account**.
  
  :::note
  The token must have `Admin` access to create new releases.
  :::

  <img alt="New Service Accounts Dialog" src="/images/service-accounts-dialog.png" width="400px"/>

  [View a larger version of this image](/images/service-accounts.png)

1. Copy the service account token and save it in a secure location. The token will not be available to view again.

  :::note
  To remove a service account, select **Remove** for the service account that you want to delete.
  :::
