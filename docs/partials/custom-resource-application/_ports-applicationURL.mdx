<li><p><code>ports.applicationUrl</code>: Links to the port-forwarded service from the admin console dashboard use this URL. <code>ports.applicationUrl</code> must match a service found in the Kubernetes Application manifest in the release.</p><p> For example, <code>ports.applicationUrl: "http://my-application-url"</code> in the KOTS Application custom resource must match <code>descriptor.links.url: https://my-application-url</code> in the Kubernetes Application manifest.</p><p>For information about how to link to a port-forwarded service from the admin console dashboard, see <a href="/vendor/admin-console-adding-buttons-links">Adding Buttons and Links</a>.</p></li>
