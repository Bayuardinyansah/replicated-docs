# policy ls

Lists enterprise policies

### Synopsis

Lists all policies that have been created

### Usage
```bash
replicated enterprise policy ls [flags]
```


| Flag                  | Type   | Description |
|-----------------------|--------|-------------|
| `-h, --help` | | help for ls |

### Examples

```bash
replicated enterprise policy ls
ID                             NAME
```
